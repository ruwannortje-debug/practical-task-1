# Practical Task 1 - Static Website Deployment

This folder contains a ready-to-upload static website for the deployment task.

## Files included
- `index.html`
- `styles.css`
- `static_url.txt` (fill in after deployment)
- `static_page.png` (replace with your own screenshot showing the deployed URL)

## How to complete deployment
1. Create a public GitHub repository.
2. Upload the files in this folder.
3. Go to **Settings > Pages**.
4. Under **Branch**, choose `main` and save.
5. Wait for deployment.
6. Copy your live URL into `static_url.txt`.
7. Open the live site, make sure the URL is visible, take a screenshot, and replace `static_page.png`.
